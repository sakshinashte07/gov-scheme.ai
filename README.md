# GovSim AI - Policy Decision Simulator

This project simulates how government policies affect society over time.

## Features
- Policy simulation (tax, subsidy, regulation)
- Multi-factor system (pollution, economy, happiness)
- Simple AI-based decision logic

## Tech Stack
- Python
- Gradio

## How it works
User inputs system values and selects a policy.
The system simulates the impact and shows updated results.
